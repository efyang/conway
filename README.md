# conway
Conway's Game of Life in Rust (using rustty)

[![asciicast](https://asciinema.org/a/boni9ekma5lrutdbpy0cldxxx.png)](https://asciinema.org/a/boni9ekma5lrutdbpy0cldxxx)

### Controls
* q => Quit
* r => Reset
* g => Generate new
